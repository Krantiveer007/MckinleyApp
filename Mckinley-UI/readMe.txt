Hi All, Welcome to Login Logout WorkFlow -> Client side implementation, please refer to below steps for starting
the Angular application on your local machine.

Note: Before following below steps make sure you have node and Angular CLI installed on your machine.
(refer to package.json for version numbers)

Steps to run Angular application : 
    1> Run Command 'npm install' from the current directory, this command will install the required 
       node_modules from node package manager.
    2> Once installation is finished, Run Command: 'ng serve' from the current directory, this will start
       the Angular application on your local machine(i.e., http://localhost:4200).
